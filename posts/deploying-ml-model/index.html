Considerations for Deploying an ML Model


✈ Deploying a machine learning model is like flying an airplane 🛩

Having no logging is like flying without a Blackbox. Any terrorist can hijack the plane without any consequences.

Having no monitoring is like flying blind in new airspace. The pilot has no idea where he’s going, and neither do you.

Having no resilient infrastructure is like flying with an unmaintained engine. The ticket is cheap but the plane could go down at any time.

You won’t board a plane with a blind pilot and a broken engine, why would you deploy a model without logging, monitoring, and resilient infrastructure?
